1) Общее
Приложение работает через API Yandex.Переводчика.
Для осуществления перевода необходимы:
 - Доступ в Интернет.
 - Бесплатный API ключ.
 - Выбранная языковая пара (языки должны отличаться).

Если одно из этих условий будет нарушено, то перевод не будет произведен.

Количество переводимых символов на базовом тарифе: 10 000 000/мес., но не более 10 000 за 1 перевод

2) Интерфейс

  2.1) Главный экран.
  Главный экран представяет собой 2 текстовых поля, с соотношением 50/50 к занимаемой области. Внизу экрана находится панель языков (2.1.1), и кнопка "Перевести" с допонительным меню (2.1.2).
    2.1.1) Панель языков.
    Панель языков находится внизу экрана, и состоит из 2-х выпадающих списков. Необходимо выбрать язык из списка, и нажать кнопку "Перевести" (2.1.2). При необходимости можно изменить направление перевода с помощью специальной кнопки, находящейся между списками.
    2.1.2) Меню.
    Меню находится справа от панели языков (2.1.1), и содержит дополнительные пункты, для перехода к различным экранам приложения. Кнопка "Перевести" для удобства вынесена наружу.
    - Очистить всё: очищает поля ввода
    - История/Избранное: открывает совмещённый экран истории и избранного
    - Настройки: открывает экран настроек

  2.2) История/Избранное.
    2.2.1) Вкладка "История" содержит список переводов. Долгий тап добавляет перевод в избранное. Свайп слева направо удаляет конкретный перевод из истории. Вы также можете удалить всю историю разом, нажав на кнопку "Удалить" в панели действий.
    2.2.2) Вкладка "Избранное" содержит только те переводы, которые вы сами туда добавите. Список обновляется при помощи кнопки "Обновить" в панели действий. Долгий тап открывает диалог редактирования перевода.
    Данные в этих вкладках независимы друг от друга. Удаление истории не влечёт за собой удаление избранного. Впрочем, справедливо и обратное уверждение.

  2.3) Настройки.
    2.3.1) Содержит список настроек приложения, ровно как и этот экран.
